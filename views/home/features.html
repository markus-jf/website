FEATURES

Dependencies and package management for developers

  • Works in any platform (Win, Linux, Mac, FreeBSD, SunOS) and support cross-building
  • Build and release your own packages (C, C++, Python, Go, Fortran) with the same tool
  • Instant upload, by version, author and channel (testing, stable...)

De-centralized

  • Run your own server in your premises, behind the firewall, with one command
  • Cache packages from other servers in yours for independent builds
  • Runs perfectly offline, even for locally creating and testing packages

Artifactory support

  • Use Artifactory to privately host your conan packages, either on-premises with Artifactory Pro, or in the cloud with Artifactory SaaS cloud
  • Host any other package for any other language or system in Artifactory too (Maven, NPM, docker, PyPI, etc).
  • Complete DevOps solution: WebUI, logging and metrics, powerful permission system, multiple authentication methods, Jenkins CI integration.

Source code and binaries

  • Conan downloads just the specific binaries you need or builds from source if there are no compatible binaries
  • Source code retrieval from any origin, git, zip downloadâ€¦ or store/snapshot the code in conan too
  • Efficient and fast, ideal for continuous integration, repeatability, testability and reliability.

Simple, flexible and powerful scripting

  • Python conanfiles to specify the stages of your package: get sources, build, package, reuse...
  • Highly configurable, manage settings (OS, compiler, version, arch) and options(static-shared, etc)
  • Use any build system. Take advantage of predefined helpers for CMake

Full control of dependencies

  • Easy dependencies configuration, overriding, conflict resolution
  • Manage transitive dependencies, private dependencies. Use semantic versioning by default or any scheme you want
  • Conditional (based on OS, compiler or user options) dependencies

Free hosting service for free software

  • Do you offer your libraries or applications for free? Enjoy free hosting in our servers
  • Both OSS and binaries, as long as they are free (gratis)
  • Full functionality forever free. Integrate nicely with Github, TravisCI, Appveyor, etc

Full open-source stack

  • Permissive MIT license, including the server. Vibrant and active OSS community
  • Zero lock-in. Stay free, easily opt-in/out to use conan without changing your tools
  • Clean python codebase allows quick and simple customization
